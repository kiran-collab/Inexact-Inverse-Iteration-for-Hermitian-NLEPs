PROJECT INDEX
=============

Inexact Inverse Iteration with Sketched Inner Solvers for Eigenvector
Refinement in Hermitian Nonlinear Eigenvalue Problems (NLEPs)

Numerical experiments accompanying the paper. Implements inexact inverse
iteration (Algorithm 4.2) with a sketch-and-precondition LSQR inner solver
(Algorithm 6.1) for Hermitian nonlinear eigenvalue problems.


SOURCE CODE
-----------
numerical_experiments.py
    Main experiments and three sensitivity studies (Section 8) on the
    loaded_string NLEP. Includes the NLEP definition, reference eigenvalue
    computation via quadratic linearisation, sketch-and-precondition LSQR,
    and inexact inverse iteration.
    Run: python numerical_experiments.py

multi_problem_experiments.py
    Multi-problem scaling study across three NLEP classes:
      - loaded_string (1D rational Hermitian, n in {100, 500, 2000})
      - hadeler       (transcendental Hermitian, n = 200, dense)
      - plate2d       (2D rational Hermitian, sparse, n = 2500)
    Compares direct (LU) vs sketched LSQR inner solvers.
    Run: python multi_problem_experiments.py
